# SETUP
## Downloading data
- download the spreadsheet linked at the top of the document
  - https://docs.google.com/spreadsheets/d/1QyAuiphFFS0CTUge2Rm8706e-NPg3lzJ/edit?usp=sharing&ouid=118325991457207812648&rtpof=true&sd=true
- download a reference album cover image
  - example: https://drive.google.com/file/d/1OT82Vg8XIlqdq9pd7iM9um-yHdjzCBP2/view?usp=drive_link 
- download or create reference Taylor Swift album cover dataset
  - example: https://drive.google.com/drive/folders/1fQMi5GJJhK_mbGcahZXQLfzitZrQsiJ-?usp=drive_link
## Uploading data
- upload reference album image cover to both files and change upload path if necessary 
- upload reference album cover dataset to IP-Adapter file and change upload path if necessary 
- run first two boxes and select the genius data spreadsheet when prompted to upload a file
