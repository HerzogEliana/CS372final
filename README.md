**What it Does**
*README.md has What it Does section that describes in one paragraph what your project does*
My project generates a playlist of Taylor Swift songs based on a given song title and then generates a few versions of an original song based on that a title then compares which has the highest similarity score, selecting that one as the playlist to be used. It then generates a cover image for the playlist, using stable diffusion and conditioning before fine-tuning it to attempt to generate an original but realistic looking album cover.
